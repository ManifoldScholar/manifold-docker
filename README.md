# Manifold Docker

## Using the helper script

```
Usage: ./manifold-docker

Commands:

    build
    compose
    stack

```

## Getting started

Build the images

`./manifold-docker build`

Run the containers 

`./manifold-docker compose up`

Test it out in your browser: 
http://docker.manifold.lvh:4000/api/v1/projects

Take it down

`./manifold-docker compose down`
